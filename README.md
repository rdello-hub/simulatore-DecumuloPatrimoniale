# Simulatore Decumulo Patrimoniale 📈
**Riccardo Dell'Orletta — Consulente Finanziario**

Un simulatore avanzato e interattivo per la pianificazione finanziaria, focalizzato sulla fase di decumulo (spesa del patrimonio). Progettato per fornire chiarezza sulla sostenibilità del patrimonio nel tempo, considerando rendimenti attesi, inflazione reale e costi di gestione.

![Anteprima del Progetto](https://placehold.co/1200x600/0b1120/f5ff85?text=Simulatore+Decumulo+Patrimoniale)

## Caratteristiche Principali ✨
- **Scenari Multipli**: Passa istantaneamente tra scenari Prudente, Base e Stress.
- **Strategie di Prelievo**: Supporta prelievi fissi, indicizzati all'inflazione o a percentuale variabile.
- **Analisi Reale vs Nominale**: Considera l'erosione del potere d'acquisto nel tempo.
- **Design Premium**: Interfaccia moderna con glassmorphism, responsive e ottimizzata per i professionisti del settore.
- **Grafica Dinamica**: Grafici personalizzati ad alta risoluzione (Canvas API) per una visualizzazione immediata del "punto di esaurimento".
- **Esportazione CSV**: Esporta i dati della simulazione per ulteriori analisi in Excel o strumenti terzi.

## Requisiti 🛠️
Il simulatore è una **Single Page Application (SPA)** "front-end only".
- Nessuna dipendenza esterna (No jQuery, No Chart.js).
- Utilizza Vanilla HTML5, CSS3 Moderno e ES6+ JavaScript.
- Funziona su qualsiasi browser moderno (Chrome, Safari, Edge, Firefox).

## Come Utilizzare 🚀
1. Clona il repository:
   ```bash
   git clone https://github.com/tuo-username/simulatore-decumulo.git
   ```
2. Apri il file `index.html` nel tuo browser.
3. Configura i parametri desiderati (Patrimonio, Anni, Rendimento, etc.).
4. Clicca su **"Calcola Simulazione"**.

## Struttura del Progetto 📂
- `index.html`: Struttura semantica e SEO-friendly.
- `style.css`: Design system professionale con variabili CSS e layout responsive.
- `script.js`: Motore di calcolo finanziario e logica di rendering dei grafici.
